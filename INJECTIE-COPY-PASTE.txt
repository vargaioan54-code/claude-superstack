  CLAUDE CODE SUPERSTACK - INJECȚIE COPY-PASTE (Windows)
═══════════════════════════════════════════════════════════════

PENTRU PRIETEN: deschide PowerShell ca Administrator (click dreapta pe
Start → "Terminal (Admin)") și dă paste la blocul de jos. Apasă Enter.

───────────────────────────────────────────────────────────────
  COPIAZĂ DE AICI ↓
───────────────────────────────────────────────────────────────

Set-ExecutionPolicy Bypass -Scope Process -Force; iwr -useb "https://raw.githubusercontent.com/USER/REPO/main/install-claude-superstack.ps1" | iex

───────────────────────────────────────────────────────────────
  ↑ PÂNĂ AICI
───────────────────────────────────────────────────────────────

⚠ ATENȚIE: Linia de mai sus nu funcționează încă — trebuie să urci
   întâi install-claude-superstack.ps1 pe GitHub (sau pe orice URL public),
   apoi înlocuiești "USER/REPO" cu URL-ul tău real.

═══════════════════════════════════════════════════════════════
  VARIANTA ALTERNATIVĂ (fără GitHub) — îi trimiți fișierul direct
═══════════════════════════════════════════════════════════════

1. Trimite-i fișierul:  install-claude-superstack.ps1
2. El îl salvează pe Desktop
3. Deschide PowerShell ca Admin și rulează:

   cd $env:USERPROFILE\Desktop
   Set-ExecutionPolicy Bypass -Scope Process -Force
   .\install-claude-superstack.ps1

═══════════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

❌ "winget not found"
   → Update Windows / instalează "App Installer" din Microsoft Store

❌ "npm not found" după instalare Node
   → Restart PowerShell (PATH-ul se reîncarcă)

❌ "claude command not found"
   → Verifică: npm root -g
   → Adaugă path-ul la PATH dacă lipsește

❌ Un MCP server pică
   → claude mcp remove <name>
   → claude mcp add ... (re-rulează linia respectivă)
